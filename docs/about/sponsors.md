## Tooling we stand on

Infrastructure Mapper is composed of open-source giants:

- [PostgreSQL] + [PostGIS] &mdash; the canonical store.
- [GDAL/OGR] &mdash; the GeoPackage build pipeline.
- [QGIS] &mdash; the desktop and field map client.
- [QField] and [Mergin Maps] &mdash; field capture.
- [Nix] + [Determinate Systems][det-sys] &mdash; the reproducible dev shell and CI.
- [mkdocs-material] &mdash; this documentation site.
- [sqlfluff], [black], [ruff], [shfmt], [shellcheck], [markdownlint],
  [cspell], [reuse] &mdash; the quality gates.

Without these projects, Infrastructure Mapper wouldn't exist. Please
sponsor them too if you can.
